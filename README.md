## 🟣 프로젝트 간단 소개

_그동안 막연히 대본만 보며 발표 준비를 하고 계셨나요?_

_그렇다면 **발표 암기 연습과 피드을 제공해주는 Presen**을 경험해 보세요!_


## 🟣 주요 기능

### 발표 자료 등록 🗒️

- 내가 발표할 ppt 이미지와 대본 등록

### 발표 연습 모드 선택 💻

- 주요 문장 가리기 or 모든 문장 보기
- 데스크탑 vs 데스크탑+모바일

### 발표 녹음 및 피드백 💯

- 발표 대본에 맞춘 실제 발화 연습 및 AI기반 피드백
